My dotfiles based on Dotbot.

# Prerequisites
- Install Git.
- Install Brew - https://brew.sh/

# Install

1. Clone this or download to `~/dotfiles`.
2. Initialize submodules.

```
git submodule update --init --recursive
```

2. Install.
```sh
./install
```
